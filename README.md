# Arquitectura de Aprendizaje Interactivo Interfaz y Progreso (AAIIP) 🎓

Plataforma educativa interactiva diseñada como una POC de alto impacto visual. El objetivo es proporcionar una experiencia de aprendizaje guiada donde el progreso se valida mediante retos de Q&A.

## 🚀 Inicio Rápido

1.  **Clonar el repositorio:** https://github.com/cluna-8/Architecture_main
2.  **Abrir `index.html`:** Al ser una aplicación Vanilla JS/HTML, puedes abrirla directamente en cualquier navegador moderno.

## 🏗️ Arquitectura

El proyecto sigue una arquitectura de **Single Page Application (SPA) ligera**.

Consulta el documento [ARCHITECTURE.md](./ARCHITECTURE.md) para más detalles técnicos.
